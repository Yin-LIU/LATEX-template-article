\section{Introduction}

This template collects the common setup used for article drafts: page layout, theorem environments, mathematics, algorithms, figures, appendices, references, and author-colored revision marks. For a final build, load the package with

\verb|\usepackage{basic_template}|.

For live preview in Texifier, use

\verb|\usepackage[texifier]{basic_template}|.

Texifier mode disables a small set of \texttt{cleveref} customizations that can conflict with the live TexpadTeX engine. In this mode, \verb|\cref| may use the default name for the \texttt{assumption} environment, and appendix section references may remain ordinary section references. For the final PDF, compile without the \texttt{texifier} option.

\subsection{Revision Marks}

Define author IDs in the preamble with \verb|\defineauthor{ID}{color}|. The optional ID selects the display color; if no ID is given, blue is used.
\begin{itemize}
    \item \verb|\added[ID1]{Text added by ID1}| \added[ID1]{Text added by ID1}
    \item \verb|\deleted[ID1]{Text deleted by ID1}| \deleted[ID1]{Text deleted by ID1}
    \item \verb|\replaced[ID1]{New text}{Old text}| \replaced[ID1]{New text}{Old text}
    \item \verb|\added[ID2]{Text added by ID2}| \added[ID2]{Text added by ID2}
    \item \verb|\deleted[ID2]{Text deleted by ID2}| \deleted[ID2]{Text deleted by ID2}
    \item \verb|\replaced[ID2]{New text}{Old text}| \replaced[ID2]{New text}{Old text}
    \item \verb|\note[ID1]{Comment by ID1}| \note[ID1]{Comment by ID1}
    \item If no author ID is assigned, the default color (blue) will be used. \added{The default color for changes is blue.}
\end{itemize}

The revision commands accept mathematical content and paragraph breaks:

\added[ID1]{
This is the first paragraph. 

This is the second paragraph.
\begin{align*}
    f(x) &= \frac{1}{2}x^2 \\
    g(x) &= x^3
\end{align*}
}

\replaced[ID1]{This is the new part}{This is the first deleted paragraph.

This is the second paragraph.
\begin{align*}
    f(x) &= \frac{1}{2}x^2 \\
    g(x) &= x^3
\end{align*}}

\subsection{References and Appendices}

Use the entries in \texttt{refs.bib} with Natbib commands:
\begin{itemize}
    \item \verb|\citet{citekey}| for textual citations;
    \item \verb|\citep{citekey}| for parenthetical citations;
    \item \verb|\cite{citekey}| for the default citation style.
\end{itemize}
For example, \citet{Cyrus2018RobustAcceleratedOptimization} and \citep{Cyrus2018RobustAcceleratedOptimization} show textual and parenthetical citations.

Appendix sections can be written inside the \texttt{appendices} environment near the end of the file. In the final build, the template aliases appendix section references through \texttt{cleveref}; in Texifier mode this alias is skipped for compatibility.

\newpage
% Bibliography section
\bibliographystyle{plainnat}
\bibliography{refs.bib}

\newpage
% Appendices section
\begin{appendices}
\section{Appendix Section Title}
% Additional appendices content here
\end{appendices}
